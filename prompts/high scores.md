# High scores

I want the game to track player high scores. Track up to 10 best scores

Data to track:
- player name
- player score
- date and time of scoring


Create a new component for listing high scores. Show the component on the main page. Sort by best score.

When the player's game ends, show a "Submit high score" button below the "Start a new game" button. 
Use a different color for this new button. If the user presses the "Submit high score", show a form with an
input field (name: name, label: "Player name") and buttons "Submit high score!" (has default focus) and "Cancel".

Store the high scores into localstorage.

After any of these buttons is pressed, take the user back to main page.
