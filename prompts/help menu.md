# Help menu

Add a "Help" button below the "Start game" button. Use a different color.

Pressing the "Help" button opens a help page. The help page should have the same background color as the game
page and font color should be white.

Help page title is "Help".

There are two sections at the moment:
- Controls
- Powerups


The Controls section describes how to play, which keys can be used. Also mention how to quit or pause the game.


The Powerups section lists all currently available powerups:
- icon
- name
- short description

After the help content. there is a button named "Back to main"
